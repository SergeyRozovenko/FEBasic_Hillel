Типи селекторів

Селектор Приклад Опис
Селектор елементу h1 { } Змінює стилі відповідно елементу
Селектор класу .warp { } Змінює стиль для усього, до чого застосован даний клас
Селектор ID #init { } Використовуєтся так же, як й селектор класу. Може бути використаний тільки один id для елементу
Селектор атрибутів a[title] { } Використовуєтся для стилізації окремих атрибутів
Комбінований селектор a[title], h1, #init { } Якщо потрібно зробити стиль для кількох класів одразу
Селектор за вкладенністю .warp a або .warp>a {} Для стилізації елементу який входит у склад іншого елементу
